# 大脑结构概览（精简版）

> 本文档从宏观视角展示人脑的核心组成部分，适合快速建立整体认知框架。

## 总体架构

人脑重约 1.4 kg，包含约 1000 亿个神经元（Neuron）和数万亿突触（Synapse）连接。
从解剖学角度可分为三大核心区域：

```
┌─────────────────────────────────────────────────┐
│                  人脑 (Human Brain)               │
│                                                   │
│  ┌──────────────┐  ┌──────────┐  ┌────────────┐  │
│  │   大脑        │  │  小脑     │  │   脑干      │  │
│  │  Cerebrum     │  │Cerebellum│  │ Brainstem   │  │
│  │  (~85%体积)   │  │(~10%体积)│  │ (~5%体积)   │  │
│  └──────────────┘  └──────────┘  └────────────┘  │
└─────────────────────────────────────────────────┘
```

## 六大核心结构

| 结构 | 英文名 | 位置 | 核心功能 | 关键联动 |
|------|--------|------|----------|----------|
| **大脑皮层** | Cerebral Cortex | 大脑最外层 | 思维、语言、感知、决策 | 丘脑（输入）、基底神经节（运动） |
| **丘脑** | Thalamus | 间脑中央 | 感觉信息中转站 | 皮层（输出）、脑干（输入） |
| **下丘脑** | Hypothalamus | 丘脑下方 | 体温、饥饿、睡眠等内稳态调节 | 垂体（激素）、边缘系统（情绪） |
| **边缘系统** | Limbic System | 大脑深部 | 情绪、记忆、动机 | 皮层（认知）、下丘脑（生理） |
| **小脑** | Cerebellum | 后颅窝 | 运动协调、平衡、运动学习 | 大脑皮层（计划）、脑干（执行） |
| **脑干** | Brainstem | 大脑底部 | 呼吸、心跳等生命维持 | 脊髓（传导）、小脑（协调） |

## 大脑皮层四大叶区

```
          ┌─────────────────┐
          │    额叶 Frontal  │ → 决策、计划、人格、语言表达
          ├─────────────────┤
          │   顶叶 Parietal  │ → 触觉、空间感知、数学
          ├─────────────────┤
          │  颞叶 Temporal   │ → 听觉、语言理解、记忆
          ├─────────────────┤
          │  枕叶 Occipital  │ → 视觉处理
          └─────────────────┘
```

## 边缘系统核心组件

| 结构 | 英文名 | 核心功能 |
|------|--------|----------|
| 海马体 | Hippocampus | 记忆形成与空间导航 |
| 杏仁核 | Amygdala | 情绪处理（尤其恐惧） |
| 扣带回 | Cingulate Gyrus | 情绪调节与注意力 |

## 脑干三部分

| 结构 | 英文名 | 核心功能 |
|------|--------|----------|
| 中脑 | Midbrain | 视听反射、运动调节 |
| 脑桥 | Pons | 睡眠、呼吸、面部运动 |
| 延髓 | Medulla Oblongata | 心跳、呼吸、血压控制 |

## 信息流简图

```
外部刺激 → 感觉器官 → 丘脑(中转) → 大脑皮层(处理)
                                         ↕
                                    边缘系统(情绪/记忆)
                                         ↕
                              基底神经节(运动决策) → 小脑(协调) → 运动输出
                                         ↓
                                    脑干(生命维持) → 脊髓 → 身体
```
