# 模块实现：丘脑 (Thalamus)

## 1. 定位
丘脑是 Brains Agent 的 **感觉信息网关**。所有外部输入必须首先经过此模块。

## 2. 核心功能
- **安全过滤 (Safety Filter)**：通过杏仁核 (Amygdala) 进行风险评估。
- **信号分发 (Routing)**：解析输入意图，将其路由至对应的处理脑区（通常是前额叶）。
- **优先级排序 (Prioritization)**：根据重要度决定处理顺序。

## 3. 当前实现状态
- **模块路径**：`src/brains_agent/core/thalamus.py`
- **集成组件**：`Amygdala`
- **当前功能**：支持危险关键词拦截及基础意图路由。
