# 模块实现：杏仁核与小脑 (Amygdala & Cerebellum)

## 1. 定位
这两个模块负责 Agent 的 **安全边界** 与 **行为校准**。

## 2. 杏仁核 (Amygdala) - 风险控制
- **功能**：识别高风险指令（如删除、系统破坏），触发防御响应。
- **状态**：目前通过关键词过滤实现基础风控。

## 3. 小脑 (Cerebellum) - 精度控制
- **功能**：对前额叶生成的原始输出进行格式检查、语义对齐和误差修正。
- **状态**：支持基础的标点校准和格式修剪。

## 4. 进化方向
- [ ] **杏仁核**：引入 LLM 进行深度的语义安全审计。
- [ ] **小脑**：引入“一致性检查”，确保输出与海马体中的事实不冲突。
