# 大脑结构详解（中等版）

> 本文档从系统层面展示人脑结构，涵盖各区域的子结构、功能细节与区域间协作关系。

## 一、总体架构

人脑从胚胎学角度分为五大脑区，成熟后形成以下层次：

```
人脑 (Human Brain)
├── 前脑 Forebrain (Prosencephalon)
│   ├── 端脑 Telencephalon
│   │   ├── 大脑皮层 Cerebral Cortex
│   │   ├── 基底神经节 Basal Ganglia
│   │   ├── 边缘系统 Limbic System
│   │   └── 白质纤维束 White Matter Tracts
│   └── 间脑 Diencephalon
│       ├── 丘脑 Thalamus
│       ├── 下丘脑 Hypothalamus
│       ├── 上丘脑 Epithalamus
│       └── 底丘脑 Subthalamus
├── 中脑 Midbrain (Mesencephalon)
│   ├── 顶盖 Tectum
│   └── 被盖 Tegmentum
├── 后脑 Hindbrain (Rhombencephalon)
│   ├── 小脑 Cerebellum
│   └── 脑桥 Pons
└── 末脑 Myelencephalon
    └── 延髓 Medulla Oblongata
```

## 二、大脑皮层 Cerebral Cortex

大脑皮层覆盖大脑表面，厚约 2-4mm，面积约 2500 cm²，通过沟回折叠大幅增加表面积。
分为**新皮质（Neocortex，6 层）和旧皮质（Allocortex，3-4 层）**。

### 2.1 四大叶区


| 叶区     | 英文名            | 位置    | 主要功能               | 关键子区                                                               | 联动结构              |
| ------ | -------------- | ----- | ------------------ | ------------------------------------------------------------------ | ----------------- |
| **额叶** | Frontal Lobe   | 中央沟前方 | 执行功能、计划、决策、人格、语言表达 | 前额叶皮质 (PFC)、布洛卡区 (Broca's Area, BA44/45)、初级运动皮层 (BA4)              | 基底神经节、丘脑、边缘系统     |
| **顶叶** | Parietal Lobe  | 中央沟后方 | 体感处理、空间认知、数学运算     | 初级体感皮层 (BA1/2/3)、后顶叶皮层 (BA5/7)                                     | 额叶（感觉运动整合）、颞叶     |
| **颞叶** | Temporal Lobe  | 外侧裂下方 | 听觉处理、语言理解、记忆       | 韦尼克区 (Wernicke's Area, BA22)、初级听觉皮层 (BA41/42)、梭状回 (Fusiform Gyrus) | 海马体、杏仁核、额叶        |
| **枕叶** | Occipital Lobe | 大脑最后部 | 视觉信息处理             | 初级视觉皮层 (V1, BA17)、视觉联合区 (V2-V5)                                    | 顶叶（空间视觉）、颞叶（物体识别） |


### 2.2 前额叶皮质 Prefrontal Cortex (PFC) — 重点展开

前额叶是人类区别于其他物种的关键区域，占皮层面积约 30%。


| 子区     | 英文名                             | 功能             | 联动        |
| ------ | ------------------------------- | -------------- | --------- |
| 背外侧前额叶 | Dorsolateral PFC (dlPFC)        | 工作记忆、认知灵活性、计划  | 顶叶、丘脑     |
| 腹内侧前额叶 | Ventromedial PFC (vmPFC)        | 价值判断、情绪调节、社会认知 | 杏仁核、海马体   |
| 眶额叶皮质  | Orbitofrontal Cortex (OFC)      | 奖赏评估、决策        | 杏仁核、纹状体   |
| 前扣带皮层  | Anterior Cingulate Cortex (ACC) | 冲突监控、错误检测、注意力  | dlPFC、杏仁核 |

## 三、间脑 Diencephalon

### 3.1 丘脑 Thalamus

- **位置**：间脑中央，两个卵圆形核团
- **功能**：几乎所有感觉信息（除嗅觉）的中转站和初步处理中心
- **主要核团**：


| 核团    | 英文名                              | 功能      | 连接                   |
| ----- | -------------------------------- | ------- | -------------------- |
| 外侧膝状体 | Lateral Geniculate Nucleus (LGN) | 视觉中继    | 视网膜 → LGN → V1       |
| 内侧膝状体 | Medial Geniculate Nucleus (MGN)  | 听觉中继    | 耳蜗核 → MGN → 听觉皮层     |
| 腹后外侧核 | Ventral Posterolateral (VPL)     | 躯体感觉中继  | 脊髓 → VPL → 体感皮层      |
| 前核    | Anterior Nucleus                 | 记忆、情绪   | 海马体 → 乳头体 → 前核 → 扣带回 |
| 背内侧核  | Mediodorsal Nucleus (MD)         | 执行功能、情绪 | 杏仁核 → MD → 前额叶       |


### 3.2 下丘脑 Hypothalamus

- **位置**：丘脑下方，第三脑室底部
- **功能**：自主神经系统的最高调节中枢，内分泌系统的枢纽
- **主要核团与功能**：


| 核团                    | 功能                |
| --------------------- | ----------------- |
| 视交叉上核 (SCN)           | 昼夜节律 / 生物钟        |
| 视上核 / 室旁核             | 分泌抗利尿激素 (ADH)、催产素 |
| 弓状核                   | 食欲调节              |
| 乳头体 (Mammillary Body) | 记忆回路（Papez 环路）    |

## 四、边缘系统 Limbic System

边缘系统位于大脑深部，是情绪、记忆和动机的核心系统。

### 4.1 核心组件


| 结构      | 英文名             | 位置         | 功能                    | 联动                          |
| ------- | --------------- | ---------- | --------------------- | --------------------------- |
| **海马体** | Hippocampus     | 颞叶内侧       | 短期→长期记忆转化、空间导航、情景记忆编码 | 内嗅皮层（输入）、前额叶（提取）、杏仁核（情绪记忆）  |
| **杏仁核** | Amygdala        | 颞叶前内侧      | 恐惧条件化、情绪标记、社会认知       | 前额叶（调节）、海马体（情绪记忆）、下丘脑（应激反应） |
| **扣带回** | Cingulate Gyrus | 大脑纵裂内侧     | 情绪调节、疼痛感知、注意力分配       | 前额叶、杏仁核、运动皮层                |
| **隔区**  | Septal Nuclei   | 基底前脑       | 奖赏、愉悦感                | 海马体、下丘脑                     |
| **穹窿**  | Fornix          | 海马体上方弧形纤维束 | 海马体主要输出通路             | 海马体 → 乳头体 → 丘脑前核            |


### 4.2 Papez 环路（记忆核心回路）

```
海马体 → 穹窿 → 乳头体 → 丘脑前核 → 扣带回 → 海马旁回 → 海马体
   Hippocampus → Fornix → Mammillary Body → Anterior Thalamic Nucleus
   → Cingulate Gyrus → Parahippocampal Gyrus → Hippocampus
```

## 五、基底神经节 Basal Ganglia

- **位置**：大脑深部白质内的灰质核团群
- **功能**：运动选择与抑制、习惯形成、奖赏学习


| 结构   | 英文名                 | 功能          |
| ---- | ------------------- | ----------- |
| 尾状核  | Caudate Nucleus     | 认知控制、目标导向行为 |
| 壳核   | Putamen             | 运动控制、运动学习   |
| 苍白球  | Globus Pallidus     | 运动输出调节      |
| 伏隔核  | Nucleus Accumbens   | 奖赏、动机、成瘾    |
| 黑质   | Substantia Nigra    | 多巴胺产生、运动启动  |
| 丘脑底核 | Subthalamic Nucleus | 运动抑制、冲动控制   |


### 三大功能环路


| 环路       | 功能        | 主要通路                         |
| -------- | --------- | ---------------------------- |
| **运动环路** | 随意运动启动与终止 | 运动皮层 → 壳核 → 苍白球 → 丘脑 → 运动皮层  |
| **联合环路** | 认知决策      | 前额叶 → 尾状核 → 苍白球 → 丘脑 → 前额叶   |
| **边缘环路** | 情感与奖赏     | 边缘皮层 → 伏隔核 → 苍白球 → 丘脑 → 边缘皮层 |

## 六、小脑 Cerebellum

- **位置**：后颅窝，脑干背侧
- **体积**：占脑总体积 ~10%，但包含全脑 50% 以上的神经元
- **功能**：运动协调、平衡、运动学习、认知精细调节


| 分区   | 英文名                 | 功能        |
| ---- | ------------------- | --------- |
| 前庭小脑 | Vestibulocerebellum | 平衡与眼球运动   |
| 脊髓小脑 | Spinocerebellum     | 肢体运动协调    |
| 大脑小脑 | Cerebrocerebellum   | 运动计划、认知功能 |


**联动**：通过小脑脚 (Cerebellar Peduncles) 与脑干连接，接收大脑皮层运动指令，整合本体感觉反馈后调整运动输出。

## 七、脑干 Brainstem


| 结构     | 英文名               | 位置      | 主要功能            | 关键核团                                                                  |
| ------ | ----------------- | ------- | --------------- | --------------------------------------------------------------------- |
| **中脑** | Midbrain          | 脑桥上方    | 视听反射、运动调节、疼痛调控  | 上丘 (Superior Colliculus)、下丘 (Inferior Colliculus)、红核 (Red Nucleus)、黑质 |
| **脑桥** | Pons              | 中脑与延髓之间 | 呼吸调节、睡眠、面部感觉与运动 | 脑桥核、三叉神经核                                                             |
| **延髓** | Medulla Oblongata | 脑干最下部   | 心率、血压、呼吸、呕吐反射   | 孤束核、疑核、迷走神经背核                                                         |


**网状结构 (Reticular Formation)**：贯穿整个脑干，调节意识水平、睡眠-觉醒周期、疼痛调控和肌张力。

## 八、主要神经递质系统


| 神经递质   | 英文名            | 产生部位                 | 功能          | 相关通路           |
| ------ | -------------- | -------------------- | ----------- | -------------- |
| 多巴胺    | Dopamine       | 黑质、腹侧被盖区 (VTA)       | 奖赏、运动、动机    | 中脑边缘通路、黑质纹状体通路 |
| 5-羟色胺  | Serotonin      | 中缝核 (Raphe Nuclei)   | 情绪、睡眠、食欲    | 中缝核 → 全脑弥散投射   |
| 去甲肾上腺素 | Norepinephrine | 蓝斑 (Locus Coeruleus) | 警觉、注意力、应激   | 蓝斑 → 全脑弥散投射    |
| 乙酰胆碱   | Acetylcholine  | 基底前脑、脑干              | 记忆、注意力、肌肉收缩 | 基底前脑 → 皮层      |
| 谷氨酸    | Glutamate      | 全脑分布                 | 兴奋性传递、学习可塑性 | 皮层内及皮层间连接      |
| GABA   | GABA           | 全脑分布                 | 抑制性传递、调节兴奋性 | 皮层中间神经元、基底神经节  |
